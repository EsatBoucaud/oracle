# Megazine Studio Board

Megazine Studio Board is the merged scheduling workspace built from the provided calendar app and template assets.

## Run locally

Prerequisite: Node.js 20 or newer

1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`
3. Create a production build with `npm run build`

## Branding notes

- `Megazine` is the primary product and company branding.
- `Infogito` is kept as a small template credit inside the app shell.
- The mascot loop is served from `public/mascot-loop.webm`.
